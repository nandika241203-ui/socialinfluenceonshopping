# socialinfluenceonshopping
Social Influence on Online Shopping using SPSS Modeler
Project Overview
This project analyzes the influence of social media platforms on online shopping behavior using IBM SPSS Modeler.
The study investigates how different social media platforms affect purchasing decisions across various respondent segments such as gender, mobile users, and university groups.

Objective
Analyze social media influence on shopping behavior.
Compare platform preferences across different segments.
Apply data mining techniques for pattern discovery.
Dataset Description
The dataset contains the following variables:
Variable	Description
Index	Unique record identifier
Question	Survey question
Segment Type	Category of respondents
Segment Description	Detailed segment information
Answer	Social media platform selected
Count	Number of responses
Percentage	Percentage of responses

Models Applied
Distribution Analysis
Used to identify frequency distribution of social media platforms.
Chi-Square Analysis
Used to determine relationships between respondent segments and platform preference.
CHAID Decision Tree
Used to identify significant predictors affecting platform influence.
Key Findings
Instagram shows strong influence among female voters.
The "None" category is highly represented among male voters.
Platform influence varies across respondent segments.
CHAID analysis identified limited predictive patterns due to dataset structure.
